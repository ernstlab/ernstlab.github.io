On this page are zip files for the conditions analyzed in the MSB paper. To open a map from the paper in DREM, save the zip file for the desired condition and unzip it. This will create a directory with five files. Copy all five files directly into the main drem directory. The files are the gene expression data, the TF-gene interaction file, the saved model, a default settings file for DREM for the condition, and a cmd file to have DREM start with the corresponding default settings file (Windows only). For more details about these data files and references to their original sources please consult the Supplementary Results.
  • AA34.zip (Amino Acid Starvation Using Only Condition Specific Binding)

  • AAmerged.zip (Amino Acid Starvation Using Condition Specific Binding and Regulatory Code Information)
  • ColdShock.zip
  • DTT.zip
  • H2O2.zip
  • Heat.zip - (Already included in the DREM download)
