Biological Data

The biological data used in the experimental results of the paper is available in two files, g27_1.txt and g27_2.txt, which contain the data from the G27 TC1 trial 4 and G27 TC1 trial 5 time series available from http://genome-www5.stanford.edu/cgi-bin/publication/viewPublication.pl?pub_no=193 (Guillemin et al., 2002). The data has been put into a format for which is usable by STEM.

Gene expression values were averaged between the two repeats. If multiple spots on the array were annotated as belonging to the same gene, then their values were first averaged. Genes were filtered if any of these criteria were met:

  • The expression level was missing at any time point in both repeats
  • The absolute log base two fold change at any time point did not exceed 0.8
  • The genes expression level between the two experiments was negatively correlated
